📍 Live Tracking Web App

A real-time live tracking web application built for learning purposes using Node.js, Express, Socket.IO, and Leaflet Maps.
This project demonstrates how real-time location sharing works between users using WebSockets.

🚀 Features:
📡 Real-time location updates
🗺️ Interactive map integration
👥 Multiple users tracking simultaneously
⚡ Instant communication using Socket.IO
📍 Live marker movement on map
🔄 Auto-update without page refresh
📱 Responsive UI

🛠️ Tech Stack:
Frontend:
HTML
CSS
JavaScript
Leaflet.js

Backend:
Node.js
Express.js
Socket.IO
